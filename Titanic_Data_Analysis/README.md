# Titanic Data Analysis

##  Project Overview

This project performs exploratory data analysis on the famous Titanic dataset. The goal is to analyze passenger data and understand the factors that influenced survival during the Titanic disaster.

Using data analysis techniques, we explore relationships between passenger attributes such as age, gender, passenger class, and survival rate.

---

##  Dataset

The dataset contains information about passengers aboard the Titanic, including:

* Passenger ID
* Name
* Age
* Gender
* Passenger Class
* Fare
* Embarked Port
* Survival Status

---

##  Objectives

The main objectives of this project are:

* Perform data cleaning and preprocessing
* Explore relationships between variables
* Analyze survival patterns
* Visualize insights using graphs and charts

---

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

##  Analysis Performed

Some of the analysis performed in this project includes:

* Survival rate by gender
* Survival rate by passenger class
* Age distribution of passengers
* Correlation between features
* Visualization of survival trends
